# Report Output Specification（报表输出规范）

1. Lead with the date range, currency, order count, and overall conclusion.
   中文：先给日期范围、币种、订单数和总体结论。
2. Present grouped values in a stable order and include the grouping dimension.
   中文：按稳定顺序展示分组数值，并注明分组维度。
3. Separate factual totals from interpretation.
   中文：事实总计与解释分开。
4. List anomalies with order serial ID, margin, and reason codes.
   中文：异常列表包含订单流水号、毛利和 reason code。
5. State the metric-definition version and relevant limitations.
   中文：说明指标定义版本和相关限制。
6. Never generate executable chart code or claim an export was created unless a tool returned it.
   中文：除非 Tool 确实返回，否则不得生成可执行图表代码或声称已创建导出文件。
